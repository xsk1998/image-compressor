# 图片压缩工具

这是一个简单易用的在线图片压缩工具,采用苹果风格设计,帮助用户快速压缩图片文件大小。

## 在线使用
- 访问：https://image-compressor-a75n.vercel.app

## 主要功能

1. 支持上传 PNG、JPG 格式图片
2. 支持自定义压缩比例
3. 实时预览压缩前后的图片效果
4. 显示压缩前后文件大小对比
5. 支持压缩后图片下载

## 技术实现
- 使用原生 HTML5 和 CSS3 构建界面
- 使用 FileReader API 读取图片
- 使用 Canvas API 实现图片压缩
- 采用 Flexbox 布局实现响应式设计
- 遵循苹果设计风格
